List of 30 HTML tags and their functions.

  1. <!DOCTYPE>=Defines the document type.
  2. <html>=Defines the html document.
  3. <title>=defines a title for the document.
  4. <body>=Defines the document's body.
  5. <h1-h6>=Defines Html headings.
  6. <p>=Dedfines a paragraph.
  7. <br>=inserts a single line break.
  8. <hr>=The hr element represents a paragraph-level thematic break.
  9. <blockquote>=Defines a section that is quoted from another source.
  10. <b>=Defines bold text.
  11. <abbr>=Abbreviation for an acronym.
  12. <cite>=Defines the title of a work.
  13. <address>=Defines contact informationn for the author or owner of a document
  14. <del>=Defines text that has been deleted from a document.
  15. <dfn>=Specifies a term that is going to be defined within the content.
  16. <em>=defines emphasized text.
  17. <i>=Defines a part of text in an alternate voice or mood
  18. <kbd>=Defines keyboard input.
  19. <ins>=Defines a text that has been inserted into a document.
  20. <q>=Defines a short quotation.
  21. <s>=Defines text that is no longer correct
  22. <strong>=Defines important text.
  23. <small>=Defines smaller text
  24. <sub>=Defines subscripted text.
  25. <pre>=Defines preformatted text.
  26. <form>=Defines an HTML form for user input
  27. <label>=Defines a label for an input element.
  28. <option>=Defines an option in a drop down list.
  29. <img>=Defines an image.
  30. <nav>=Defines navigation links.
